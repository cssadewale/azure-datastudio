❓ Azure DataStudio v7 — Help & Documentation
🔍 Search All Tables
Search for a value across every column in every loaded table simultaneously. Results show the table, column, and matching row.
Enter a search term above and press Enter or click Search.
▶■ Query Macros
Macros are sequences of SQL statements that run automatically one after another. Record a macro by executing queries while recording is active, then replay them any time.
⇩ Advanced Export Options
📊 Column Analyser
Select a table and column.
🕐 Schema Change History
Every CREATE TABLE, DROP TABLE, ALTER TABLE, and CREATE INDEX statement is logged here automatically.
🔠 Regex Tester
Test regular expressions against sample text before using them in SQL GLOB or LIKE patterns. SQLite uses GLOB (case-sensitive) and LIKE (case-insensitive with % and _).
📄 Query Templates
Parameterised query templates. Select a template, fill in table and column names, and generate the SQL automatically.
